Ventana login

La ventana tiene un botón para iniciar sesión y abrir la ventana principal, que también tiene su propia página de ayuda.

Inicio en la aplicación

  • La aplicación comienza solicitando un usuario y contraseña con el que se conectará a Twitter. Hasta que el usuario no se conecte correctamente, no podrá continuar en la aplicación
  • Se permite al usuario guardar el usuario y contraseña para usos posteriores.
  • Aunque haya una contraseña guardada, siempre se podrá iniciar sesión como un usuario diferente cuando se desee.
